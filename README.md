1. Klient- og serverdrift

Installere og sette opp operativsystemer (Windows, Linux, macOS).

Administrere brukerkontoer, grupper og rettigheter.

Sette opp servere for f.eks. filer, printere, web eller e-post.

Utføre oppdateringer og sikkerhetskopiering.

Overvåke systemer for feil og ytelse.

 2. Nettverk

Bygge og konfigurere lokale nettverk (LAN).

Forstå og bruke IP-adresser, subnett, DNS og DHCP.

Konfigurere rutere, svitsjer og brannmurer.

Teste nettverkstilkoblinger og feilsøke problemer.

Bruke verktøy som Wireshark eller Ping/Traceroute for feilsøking.

3. IT-sikkerhet

Lære om trusler, virus, phishing og brannmur-konfigurasjon.

Implementere sikkerhetsrutiner og passordpolicyer.

Forstå kryptering, backup, og tilgangskontroll.

Utføre sikkerhetskopiering og gjenoppretting.

 4. Brukerstøtte og service

Gi brukerstøtte til elever, lærere eller “kunder”.

Feilsøke maskinvare og programvare.

Kommunisere profesjonelt og forklare tekniske problemer på en enkel måte.

Dokumentere arbeid, lage veiledninger og rutiner.

5. Maskinvare og systemforståelse

Montere og vedlikeholde PC-er og periferiutstyr.

Forstå komponenter: CPU, RAM, harddisk, hovedkort, nettverkskort osv.

Lære grunnleggende feilsøking på maskinvare.

☁️ 6. Skytjenester og virtualisering

Bruke Microsoft 365 / Google Workspace til drift og administrasjon.

Lage virtuelle maskiner (VMware, VirtualBox, Hyper-V).

Forstå hvordan skyteknologi (Azure, AWS) fungerer.

7. Dokumentasjon og prosjektarbeid

Skrive tekniske rapporter og IT-dokumentasjon.

Jobbe med prosjekter i grupper, ofte med reelle problemstillinger.

Følge IKT-serviceprosesser (f.eks. ITIL-inspirerte metoder).

 8. Programfag og tverrfaglig arbeid

Samarbeide med andre linjer (for eksempel med design og utvikling).

Forstå hvordan programvare og systemer brukes i bedrifter.

Gjøre tverrfaglige oppgaver og praksis i bedrift (YFF).
